## 📁 프로젝트 소개
> *"쉽고 간편한, 이용자 중심의 인트라넷"*<br/>
> <br/>
> 팀원들의 재직 경험을 살려, 간편한 기능 중심의 인트라넷을 만들어 보고자 시작한 프로젝트입니다.<br/>
> 기능을 크게 마이페이지, 공지사항 게시판, 사진 게시판, 근태관리, 개인일정관리, 결재상신, 회의실 예약으로 나눌 수 있습니다. <br/>
